有序数组的平方

题目介绍

给定一个按非递减顺序排序的整数数组 A，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。

示例 1：

输入：[-4,-1,0,3,10]
输出：[0,1,9,16,100]
示例 2：

输入：[-7,-3,2,3,11]
输出：[4,9,9,49,121]

解题思路

我们可以定义一个和原数组大小一样的数组，然后定义两个指针指向头尾，然后将数值大的放到数组结尾，直到指针相遇

length = len(A)
result = [0] * length
p = length - 1
p1 = 0
p2 = length -1
while(p1 <= p2):
	if A[p1] * A[p1] >= A[p2] * A[p2]:
		result[p] = A[p1] * A[p1]
        p1 += 1
    else:
        result[p] = A[p2] * A[p2]
        p2 -= 1
   	p -= 1
return result
