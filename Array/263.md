丑数

题目介绍

编写一个程序判断给定的数是否为丑数。

丑数就是只包含质因数 2, 3, 5 的正整数。

示例 1:

输入: 6
输出: true
解释: 6 = 2 × 3
示例 2:

输入: 8
输出: true
解释: 8 = 2 × 2 × 2
示例 3:

输入: 14
输出: false 
解释: 14 不是丑数，因为它包含了另外一个质因数 7。

解题思路
从丑数的定义中我们可以知道，丑数对2，3，5一直除，最终会得到1,所以可以根据最终结果是否等于1来判断是否是丑数

if num <= 0:
	return False
for i in [2,3,5]:
	while(num % i == 0):
		num = num / i
return num == 1
